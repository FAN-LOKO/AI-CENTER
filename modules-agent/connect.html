Получение токена через BotFather

  • Откройте @BotFather в Telegram.
  • Отправьте команду /mybots или /newbot, если создаёте нового.
  • Выберите нужного бота из списка.
  • Нажмите API Token.
  • Скопируйте токен и вставьте его в поле ниже.

После подключения токен будет сохранён в настройках модуля и использован для связи с Telegram.
